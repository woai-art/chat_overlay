  🔐 БЫСТРАЯ НАСТРОЙКА GOOGLE OAUTH - 5 МИНУТ
═══════════════════════════════════════════════════════════════

📌 ЧТО НУЖНО СДЕЛАТЬ:

1️⃣  Откройте: https://console.cloud.google.com/
    (Войдите в свой Google аккаунт)

2️⃣  Создайте проект:
    ▶ Нажмите выпадающий список вверху
    ▶ "НОВЫЙ ПРОЕКТ" → название: "YouTube Chat Parser"
    ▶ "Создать"

3️⃣  Включите YouTube API:
    ▶ В поиске введите: "YouTube Data API v3"
    ▶ Нажмите "ВКЛЮЧИТЬ"

4️⃣  Настройте OAuth Consent Screen:
    ▶ Меню → "APIs & Services" → "OAuth consent screen"
    ▶ Выберите "External" → "CREATE"
    ▶ Заполните:
       - App name: YouTube Chat Parser
       - User support email: ваш email
       - Developer contact: ваш email
    ▶ "SAVE AND CONTINUE"
    
    ▶ Scopes → "ADD OR REMOVE SCOPES":
       ✅ YouTube Data API v3 → .../auth/youtube.readonly
       ✅ YouTube Data API v3 → .../auth/youtube.force-ssl
    ▶ "UPDATE" → "SAVE AND CONTINUE"
    
    ▶ Test users → "ADD USERS":
       - Добавьте свой email (Gmail от YouTube)
    ▶ "ADD" → "SAVE AND CONTINUE"
    
    ▶ Summary → "BACK TO DASHBOARD"

5️⃣  Создайте OAuth Client ID:
    ▶ Меню → "Credentials"
    ▶ "+ CREATE CREDENTIALS" → "OAuth client ID"
    ▶ Application type: "Desktop app"
    ▶ Name: YouTube Chat Parser
    ▶ "CREATE"

6️⃣  Скачайте credentials:
    ▶ Нажмите "DOWNLOAD JSON"
    ▶ Сохраните как: client_secret.json
    ▶ Поместите в папку: D:\vMix\liveChat\chat_overlay\

═══════════════════════════════════════════════════════════════

✅ ГОТОВО! Теперь запустите:
   - START.bat → опция 2 (OAuth авторизация)
   - Или AUTHORIZE_YOUTUBE.bat
